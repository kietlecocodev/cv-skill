# ATS and Recruiter Screening References

Keyword maps and best-practice notes for making tech CVs easier for applicant tracking systems and recruiters to understand.

These references do not claim access to proprietary ATS scoring formulas. They focus on truthful keyword matching, parseable formatting, clear evidence, and recruiter-readable impact.

## How to Use

1. Extract must-have and preferred requirements from the target JD.
2. Compare them against the candidate's real experience.
3. Use role keyword maps to identify missing but truthful terms.
4. Use best-practice docs to improve formatting and evidence.
5. Avoid keyword stuffing and unsupported claims.

## Safety Rules

- Never add skills the candidate cannot defend in an interview.
- Prefer exact JD terminology when it truthfully matches experience.
- Use standard section headings and readable formatting.
- Treat ATS advice as probabilistic, not guaranteed.
