# CV Template References

Reconstructed CV templates and reusable section patterns for tech-focused applications.

These files are not copied templates. They synthesize patterns from public resume examples, recruiter guidance, hiring-manager advice, and community review discussions. Source links and notes live in `sources.md`.

## How to Use

1. Pick the closest role-family template from `templates/`.
2. Use `patterns/` to improve bullets, projects, skills, and seniority signals.
3. Adapt structure to the candidate's real experience and target JD.
4. Do not invent metrics, employers, tools, education, or credentials.
5. Treat these references as guidance, not proof that one template guarantees interviews.

## Reference Boundaries

- Public sources are cited and summarized.
- Full source CVs or proprietary templates are not copied.
- Examples inside this folder are reconstructed for this plugin.
- Community sources are supporting signal, not sole authority.
